\# Telco Customer Churn \& Retention Analytics



\## Overview

This is a backend mastery portfolio project demonstrating end-to-end data skills (SQL, Python, Power BI) for analyzing telco customer churn. Aligns with PL-300 objectives.



\## Project Goals

\- Analyze churn drivers and retention strategies.

\- Showcase data modeling, ETL, and advanced analytics.



\## Tech Stack

\- Database: Supabase (PostgreSQL)

\- ETL: SQL

\- Analytics: Python

\- Visualization: Power BI
